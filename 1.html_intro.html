Html 학습하기

h태그는 제목을 의미하고, 위 아래로 약간의 여백이 삽입된다. 또한 bold처리 됨.

검색엔진에서 h태그는 중요한 키워드로 활용된다.

p태그는 단락을 의미하고, 줄바꿈이 되며, 태그 위아래로 약간의 여백이 삽입된다.

html 태그 내에서는 엔터를 치더라도 웹페이지에서 줄바꿈이 되지 않는다.

html 태그내에서는 엔터를 치더라도 줄바꿈이 되지 않고,


줄바꿈을 하려면 br태그를 사용해야 한다.

또한, 여러개의 공백도 하나의 공백으로 처리된다.

텍스트의 형식화

bold 태그는 글자를 굵게 처리한다.

strong 태그는 글자를 굵게 처리하며, 의미상으로도 강조하는 태그이다.

italic 태그는 글자를 기울임 처리한다.

em 태그는 글자를 기울임 처리하며, 의미상으로도 강조하는 태그이다.

underline 태그는 글자에 밑줄을 긋는다.

mark 태그는 글자에 형광펜으로 밑줄을 긋는 효과를 준다.

del 태그는 삭제된 텍스트를 표현한다.

sub 태그는 아래 첨자아래에 작은 글씨를 표현한다.

sup 태그는 위 첨자위에 작은 글씨를 표현한다.

Html에서의 엔티티(특수기호) 사용

1. 공백문자 :           (공백테스트)

2. less than 기호 : <

3. greater than 기호 : >

4. 앰퍼샌드 : &

5. 쌍따옴표 : "

6. 홑따옴표 : '

div와 span 태그

div 태그는 블록 레벨 요소로 블록 단위로 구분되며 줄바꿈이 발생한다.

span 태그는 인라인 요소로 글자 단위로 구분되며 줄바꿈이 발생하지 않는다.

이 문장에는 span 태그가 포함되어 있습니다. 인라인 요소이므로 줄바꿈이 발생하지 않습니다.
